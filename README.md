# PySquarePuzzle

Prototype of a two-player puzzle game

Player starts in either top left or bottom right corners.

Players race to land on the "goal square" whilst matching the colour. 

## Player 1 Controls:
Left = Left Arrow
Right = Right Arrow
Up = Up Arrow
Down = Down Arrow

## Player 2 Controls:
Left = "a" Key
Right = "d" Key
Up = "w" Key
Down = "s" Key
